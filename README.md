# Banner Command: The Banners of Coronal
A single-file browser RPG — FF6 party-splitting, Ogre Battle squad combat, FFT-scale battlefields.

## Play
Open `dist/banners-of-coronal.html` in any browser. That's it.

## Deploy to Hostinger
Upload `dist/banners-of-coronal.html` via File Manager or FTP to `public_html/` (rename to `index.html` for a clean URL). Saves use the player's browser localStorage.

## Develop
Edit files in `src/`, test with `python3 -m http.server` (open `index.html`), then `node build.js` to produce the deployable file.

See `CLAUDE.md` for architecture and backlog; `docs/strategic-analysis.md` for the design bible.
